•	Why should we structure HTML semantically for something like a login form? 
o	Semantic HTML defines the page by well understood tags that define what the element will look like and what it is used for.

•	What are some benefits of everyone using similar semantic HTML? 
Semantic elements have a defined use and expected display.  Reading someone else’s HTML code should be easy to understand how the web page will appear.
o	
•	What are some disadvantages? 
o	Elements may not behave as intended in all browsers and on different platforms.
